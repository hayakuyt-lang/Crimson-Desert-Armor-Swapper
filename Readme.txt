Crimson Desert — Armor Transmog
Make any armor look like any other armor — stats stay unchanged.
What it does
Select any armor piece you equip, choose what you want it to look like, and generate a JSON patch file. No coding required.
Requirements

Crimson Desert installed on Steam
CD JSON Mod Manager — Download here

How to use

Run HexeMarie_ArmorSwapper.exe
Click Extract & Load — auto detects your game
Pick a category (Chest / Gloves / Boots / Helm / Cloak)
Select the armor you equip (Source)
Select the armor you want it to look like (Target)
Click Add This Swap
Repeat for other pieces if needed
Click Download JSON Patch
Drop the JSON file into CD JSON Mod Manager → Apply → Launch game

After game updates
Just reopen the app and click Extract & Load again — it always reads fresh from your updated game files.
